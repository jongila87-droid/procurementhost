Pusat Transparansi Keamanan 🛡️

Keamanan website & data Anda adalah prioritas kami. Di sini kami jelaskan secara terbuka bagaimana kami melindunginya — tanpa istilah teknis yang membingungkan.

Perlindungan Berlapis

Bagaimana kami menjaga Anda

🔐

Enkripsi SSL Gratis

Setiap website otomatis mendapat sertifikat SSL — koneksi terenkripsi (HTTPS) sehingga data pengunjung aman dari penyadapan.

💾

Backup Otomatis

Pencadangan rutin agar website Anda bisa dipulihkan jika terjadi kesalahan, serangan, atau kehilangan data.

🛡️

Perlindungan DDoS

Infrastruktur pusat data kami menyaring lalu lintas berbahaya, menjaga website tetap online saat ada serangan.

🔥

Firewall Server

Lapisan firewall membatasi akses tidak sah ke server, menghalau upaya peretasan sejak gerbang awal.

📡

Pemantauan 24/7

Sistem kami dipantau sepanjang waktu. Anomali terdeteksi dan ditangani secepat mungkin.

🏢

Pusat Data Tepercaya

Server ditempatkan di pusat data dengan keamanan fisik, daya cadangan, dan jaringan andal.

🔒

Data Anda milik Anda

Kami tidak menjual data pribadi Anda. Kata sandi disimpan dalam bentuk teracak (hash), bukan teks biasa, dan akses ke data dibatasi hanya untuk keperluan layanan. Selengkapnya di Kebijakan Privasi kami.

Komitmen & pelaporan

🤝

Komitmen Kami

Jika terjadi insiden keamanan yang memengaruhi data Anda, kami berkomitmen untuk memberi tahu Anda secara jujur dan tepat waktu, beserta langkah yang kami ambil.

📨

Menemukan Celah?

Kami menghargai peneliti keamanan. Jika Anda menemukan potensi kerentanan, laporkan secara bertanggung jawab melalui halaman Kontak kami.

Apakah hosting saya sudah otomatis aman?

Ya, perlindungan dasar (SSL, firewall, anti-DDoS, backup) aktif secara default. Anda tetap disarankan memakai kata sandi kuat & memperbarui aplikasi Anda.

Seberapa sering backup dilakukan?

Frekuensi backup bergantung pada paket Anda (mis. harian pada paket Bisnis ke atas). Detailnya tertera di halaman Harga.

Apakah saya bisa mengaktifkan keamanan tambahan?

Tentu. Anda dapat mengelola SSL, kata sandi, dan pengaturan keamanan akun dari Dashboard Anda.

Hosting yang aman sejak hari pertama

Mulai dengan SSL gratis, backup, dan perlindungan berlapis — tanpa biaya tambahan.
